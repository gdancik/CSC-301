AngularJS Basics

Some of the angular directives used are below:
  • ng-init sets the initial placeholder value in the first text box
  • ng-model is used to bind the text box values to the angular variables userString and userColor
  • ng-bind is used to bind the value of usercolor to the last paragraph
  • Angular expressions (such as {{userColor}}) are used to bind angular data to HTML, including the background-color of one of the div elements.
